# NIEVA Python-Mini-Bootcamp.

__En este ejercicio de semana 2 podremos:__

* Escribir condicionales __if/elif/else__ para que un programa reaccione
  de forma diferente según los datos que reciba.
* Crear bucles __for__ y __while__ para repetir acciones sin duplicar código.
* Controlar el flujo dentro de un bucle con __break__ y __continue__.
* Leer un mensaje de error de Python y entender qué salió mal y en qué línea.
* Construir un juego interactivo completo que corre sin errores.
